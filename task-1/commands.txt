git basic commands:

1. git init - initialize local repository
2. git remote -v - to check remote repository added or not
3. git remote add origin <remote_repo_url> - to add remote repository to local repository
4. touch <file_name> - create new file_name
5. git add . - stage all files
6. git add <file_name> - stage specific file
7. git commit -m <commit_msg> - save file in remote repository with commit msg
8. git branch - to check, currently in which branch
9. git branch -M main - to move to main branch
10. git push -u origin main - to upload local commits to remote repository. (-u sets upstream(local main <-> remote main), next time, we can just run 'git push')

 if main branch has no commit reference(HEAD) means, we cannot create another new branch.
 after commiting in main branch, 

11. git branch feature-branch - create new branch
12. git checkout feature-branch -  switch to new branch 
13. git checkout -b feature branch -  single command to create new branch and switch to that branch.
